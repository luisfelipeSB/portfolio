Highly Available App with MFA

A Distributed Systems project
CODE

A project for my Distributed Systems class: a highly available (dummy) application running on replicated servers and databases, with automatic failover and strong user authentication via an authenticator app with TOTPs. The servers were configured in a master-slave system using Node.js + Express, while the databases were configured in a master-master system using Postgres + Bucardo. VMs and VNets implemented in Azure. For more information, see the code.
