SNIRH21

A project for NTT Data

TRY IT!

An IoT sensor monitoring web application I contributed to as an intern web developer while at NTT Data. We used Lando to develop a Drupal website connected to existing sensor networks at the Tagus river and its tributaries (within Portugal), complete with features for monitoring and analyzing the collected data. The SNIRH21 project was comissioned by Portugal's Environmental Protection Agency (Agência Portuguêsa do Ambiente - APA). It was a multi-year project encompassing several teams and I was fortunate enough to contribute to it as a front-end developer for two months.

Pull Prizes

A university project

A computer game developed in my artificial intelligence class where the board is a graph (made of nodes and edges), and you must pull prizes in the board towards your home node to win. You can play against your computer running the minimax algorithm, optionally with forward pruning. You can also against another human if you want. Written in p5.js.
